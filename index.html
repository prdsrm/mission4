Introduction

Ce rapport présente l'analyse des données de l'entreprise KDOU, réalisée en trois missions distinctes. Notre objectif était d'explorer la structure de la base de données, d'exploiter ces données à l'aide d'une interface low code, et d'effectuer des analyses approfondies en utilisant le langage SQL.

Chaque mission a apporté des éclairages différents sur les opérations de KDOU, permettant une compréhension globale de ses activités, de sa clientèle, et de ses produits.

Structure du Rapport

Ce rapport est divisé en plusieurs sections :

  • Mission 1 : Étude de la structure des données
  • Mission 2 : Exploitation de la base de données avec une interface low code
  • Mission 3 : Analyse approfondie avec le langage SQL
  • Conclusion : Synthèse des résultats et perspectives
